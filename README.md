# Universal Document Restoration Tool

A notebook-based tool that converts damaged, scanned, stained, or aged document images into clean, typeset PDFs that look like freshly retyped documents.

The tool uses vision-based OCR to read document images and reconstruct the text into a formatted PDF for improved readability and archival use.

# Overview

Historical documents, field notebooks, scans, and degraded archives often contain:

stains

faded ink

uneven scanning

handwriting or irregular formatting

This tool restores readability by:

Reading the document image with vision-based OCR

Extracting the text content

Reconstructing the document as a clean, typeset PDF

The result is a legible archival copy while preserving the original text content.

# Features

Restores text from scanned or damaged documents

Supports common image formats (JPEG, PNG, TIFF, BMP, WebP)

Produces clean, formatted PDF output

Optionally saves raw OCR transcription

Allows manual correction without re-running OCR

Supports batch restoration of entire folders
